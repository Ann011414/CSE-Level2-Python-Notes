# CSE-Level2-Python-Notes
## 仓库介绍 | Repository Intro
本仓库存放全国计算机二级 Python 备考代码与配套笔记，学习素材源自 B 站博主「逊哥带你学计算机」。所有知识点均以注释形式写在代码内部，便于刷题与考前复习。
内容覆盖 Python 基础语法、选择填空考点、上机编程真题，后续会持续更新补充。
